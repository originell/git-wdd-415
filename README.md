GIT Unterricht WDD 415 
======================

Wichtige Kommandos
------------------

* `$ git init`  
  Erstellt ein neues Repository.
* `$ git add DATEINAMEN...`  
  Fügt Dateien in den Staging Bereich (für den nächsten Commit).
* `$ git commit`  
  Committed Dateien aus dem Staging Bereich in die History.
* `$ git log`  
  Zeigt die History an.
